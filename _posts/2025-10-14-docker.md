### Cleanup Docker cache

```bash
$ docker builder prune --all --force
```

### Run GUI apps from a container

```bash
$ xhost +local:docker
$ docker build -t my-firefox .
$ docker run --rm -it -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix my-firefox:latest
$ xhost -local:docker
```
